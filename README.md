# Khairon Construction Website

Bu loyiha **Khairon Construction** kompaniyasi uchun yaratilgan responsive veb-sayt bo‘lib, React.js va React-Bootstrap asosida ishlaydi. Sayt 3 tilda ishlash imkoniyatiga ega: **O‘zbekcha (UZ), Ruscha (RU), Inglizcha (EN)**.

## Demo
> Demo havolasi (agar mavjud bo‘lsa)  
> [https://your-demo-link.com](https://your-demo-link.com)

## Texnologiyalar
- React.js
- React-Bootstrap
- Framer Motion (animatsiyalar uchun)
- React Icons (ikonalar)
- React Toastify (form xabarlarini bildirish uchun)
- CSS / SCSS

## Xususiyatlari
- **Responsive dizayn**: Desktop va mobil uchun moslashgan.
- **Navbar**:
  - 3 tilda ishlovchi menyu
  - Desktop va Mobile versiyasi
  - Scroll bilan active section ko‘rsatadi
- **Services**:
  - 6 xil xizmatlar bo‘limi
  - React Icons bilan ishlovchi ikonalar
  - Multilingual (UZ/RU/EN)
- **About**:
  - Tajriba va bajarilgan loyihalar sonini animatsion ko‘rsatish
  - Yandex xarita integratsiyasi
- **Contact**:
  - Ishonchli form yuborish
  - Xabar yuborilganda bildirishnoma (`toast`) ko‘rsatadi
  - Aloqa ma’lumotlari (manzil, telefon, email, ish vaqti)
- **Footer**:
  - Tezkor havolalar
  - Ijtimoiy tarmoqlar ikonasi
